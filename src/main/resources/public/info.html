Faq


How to install

  • Download and install Fabric.
  • Download meteor from this website.
  • Extract the .zip file and put the jar inside the mods folder.
  • Meteor needs Fabric API, so download it for the version you want to run and put it in your mods folder.

Older versions

If you want to connect to an older server you can use multiconnect.
